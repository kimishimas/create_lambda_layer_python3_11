# create_lambda_layer_python3_11

## Description

- Install Python 3.11
- Create s3 Bucket.
- Create and publish a Lambda Layer
- Upload the Lambda Layer to s3 Bucket 

## Usage

Please edit requirements.txt.

```
./create_lambda_layer_python3_11.sh <BUCKET_NAME> <LAYER_NAME>
```
